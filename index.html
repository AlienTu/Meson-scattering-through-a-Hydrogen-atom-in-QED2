Meson scattering through a QED2 hydrogen atom

Classical real-time laboratory for a one-atom sector of the bosonized two-flavour massive Schwinger model. The default experiment uses \(L=200\), \(\Delta x=0.1\), \(g=1.5\), \(\kappa_1=100\), \(\kappa_2=1\), builds a static atom near \(x=0\), and records signals at \(x_L=-20\), \(x_R=20\).

The atom background is built with the same half-domain Newton + mirror construction used in the Hydrogen-formation-quench spectrum page. The internal mode solver now uses the same reference-style block shifted inverse iteration rather than Lanczos checkpoint diagonalization.

Model

\[\partial_t^2\phi_i-\partial_x^2\phi_i+g^2(\phi_1+\phi_2)+\sqrt{\pi}\,\kappa_i\sin(2\sqrt{\pi}\phi_i)=J_i(t,x),\quad i=1,2.\] \[x\in[-L/2,L/2],\qquad L=200,\qquad \Delta x=0.1\text{ by default}.\]

Static atom construction uses half-axis symmetry. The fluctuation eigenproblem is solved afterwards on the full spatial domain. The threshold is computed from the vacuum \(2\times2\) mass matrix; shifted inverse iteration samples the low spectrum near and below that threshold.

Incoming meson

Default mode is none, so the internal atom motion can be studied without an incoming meson. For scattering experiments, switch to packet or continuous source. Open radiation boundaries are used by default; exact transparent boundaries for massive Klein-Gordon waves are nonlocal, so late-time boundary contamination should still be checked by increasing L.
